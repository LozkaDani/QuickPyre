### Installation

A step-by-step guide to launching the "system"

1.  Clone the repo:
    ```bash
    git clone https://github.com/LozkaDani/QuickPyre.git
    ```
2. Cd to project's path:
   ```bash
   cd QuickPyre
   ```

3. Cd to kernel:
    ```bash
    cd kernel
    ```
4. Run core.py:
   ```bash
   python shell.py
   ```
